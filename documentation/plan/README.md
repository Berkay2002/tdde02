# Plan Index

Central index for Epic planning documents generated by prompts.

- Location pattern: `documentation/plan/{epic-name}/`
  - `epic.md` — Product Requirements Document (PRD)
  - `arch.md` — Epic Architecture Specification

## How To Add A New Epic

1. Run the PRD prompt:
   - Prompt file: `.github/prompts/breakdown-epic-pm.prompt.md`
   - Output path: `documentation/plan/{epic-name}/epic.md`
2. Run the Architecture prompt:
   - Prompt file: `.github/prompts/breakdown-epic-arch.prompt.md`
   - Output path: `documentation/plan/{epic-name}/arch.md`
3. Link the new epic below.

## Conventions

- Use kebab-case for `{epic-name}` (e.g., `camera-permissions-hardening`).
- Keep PRD and Architecture in the same folder.
- Cross-link to relevant implementation docs under `documentation/implementation/` when applicable.

## Example Entry

- Camera Permissions Hardening
  - PRD: `documentation/plan/camera-permissions-hardening/epic.md`
  - Architecture: `documentation/plan/camera-permissions-hardening/arch.md`
